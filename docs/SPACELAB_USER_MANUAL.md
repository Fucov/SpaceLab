# SpaceLabOS 双屏联动使用手册

> 适用版本：LightRAG v2.x | 更新日期：2026-05-17

---

## 1. 系统概述

SpaceLabOS（太空实验室智能操作系统）是 LightRAG 内置的双屏联动交互演示系统，模拟天宫空间站六大实验舱的实时监控与智能助手对话场景。

### 1.1 双屏架构

| 界面 | 访问地址 | 核心角色 |
|------|----------|----------|
| **演示大屏** | `/#/spacelab/main` | 指挥决策中心，实时监控全站状态 |
| **平板终端** | `/#/spacelab/tablet` | 现场操作终端，AI 对话与 HITL 授权 |

两屏通过 Zustand 共享状态（舱体数据、告警日志、实验进度），平板端的操作实时同步到大屏。

### 1.2 六大实验舱

| 舱体 | 英文 ID | 典型实验 |
|------|---------|----------|
| 生命科学舱 | `life-science` | 拟南芥生长、细胞培养 |
| 流体物理舱 | `fluid-physics` | 毛细管两相流、表面张力 |
| 材料实验舱 | `material-exp` | 金属玻璃凝固、晶体生长 |
| 燃烧科学舱 | `combustion` | 微重力火焰传播、Soot 生成 |
| 对地观测舱 | `earth-observe` | 高光谱遥感、大气成分 |
| 生物技术舱 | `bio-experiment` | 蛋白质电泳、DNA 提取 |

## 2. 演示大屏使用指南

**入口**：登录后点击导航区「进入演示大屏」，或直接访问 `/#/spacelab/main`

### 2.1 页面布局

```
┌──────────────────────────────────────────────────────────────┐
│ 顶部导航栏  天宫空间站 · 实验舱监控中心    系统在线           │
├────────────┬─────────────────────────────┬────────────────┤
│ 左侧栏 20% │      中央栏 58%             │ 右侧栏 22%     │
│            │                            │                │
│ ● 算力池   │  [实验舱阵列] 或 [舱体详情] │ ● 全站环境参数  │
│   监控指标  │                            │ ● 全局资源仲裁  │
│ ● 告警日志 │                            │                │
│   (200条)  │                            │                │
└────────────┴─────────────────────────────┴────────────────┘
```

### 2.2 左侧栏

#### 2.2.1 算力池与智能体调度

- **CPU / GPU / 内存** 实时使用率（百分比 + 温度）
- **网络带宽** 使用情况
- **智能体调度**：并发任务数、推理延迟、Token 吞吐率
- **资源锁列表**：显示当前被占用的物理设备（机械臂/离心机）和 LLM 算力

#### 2.2.2 告警日志

- 默认显示全部级别（INFO / WARN / ERROR）
- 点击标签可筛选：`全部` `告警` `错误`
- 每条日志包含：时间戳、级别、来源、消息内容
- 最多保留 200 条，自动淘汰旧日志
- **实时更新**：新产生的告警立即显示在顶部

### 2.3 中央栏

#### 2.3.1 舱体阵列视图（默认）

显示所有 6 个实验舱的状态卡片，按「运行中 > 异常 > 暂停 > 完成 > 待机」排序。

每个卡片包含：
- 舱体图标 + 名称 + 当前任务
- 状态指示灯（彩色圆点 + 文字标签）
- 运行时显示：当前步骤名称 + 进度百分比
- 传感器数据：温度、气压、功率
- 进度条 + ETA

**点击任意卡片** → 进入舱体详情页

#### 2.3.2 舱体详情页

点击卡片后展开详情，包含：

1. **传感器网格**：温度 / CO₂ / 湿度 / 气压 / 功率（共 5 项）
2. **DAG 步骤流程图**：横向 SVG 流程图，实时显示各步骤状态
   - 串行步骤（蓝实线）和并行步骤（绿色虚线）
   - 节点颜色：运行中=蓝、已完成=深蓝、异常=红、等待=灰
   - 运行中节点带脉冲动画圆点
3. **历史实验列表**：点击可查看完整实验数据和图表
4. **任务队列**：显示排队中的任务及优先级

**点击「返回阵列表」** → 回到卡片视图

### 2.4 右侧栏

#### 2.4.1 全站环境参数

- 舱温 / 湿度 / 气压 / 噪声
- 带趋势箭头（↑ 上升、↓ 下降、— 稳定）

#### 2.4.2 全局资源仲裁

- 算力分配流向图（桑基图风格）
- 显示各舱体分配的 CPU / GPU / 内存比例

## 3. 平板终端使用指南

**入口**：登录后点击「进入平板终端」，或直接访问 `/#/spacelab/tablet`

### 3.1 页面布局

```
┌────────────────────────────────────────────────────┐
│ 顶部栏  天宫智能助手    [上传文档] [紧急终止] [在线]  │
├──────────────┬───────────────────────────────────┤
│ 左侧栏 260px │  主聊天区（独立滚动）                 │
│              │                                    │
│ ● 会话标签   │  [会话标题栏]                        │
│   新建实验   │  [消息列表 + DAG 编辑器]               │
│   新建知识   │  [Markdown 渲染 + 思考折叠]           │
│              │                                    │
│ ● 活跃实验   │                                    │
│   (底部)    │                                    │
├──────────────┤  ────────────────────────────── │
│              │  [输入框]  [发送]                    │
│              │  [打开实验步骤设计器]                  │
└──────────────┴───────────────────────────────────┘
```

### 3.2 会话管理

#### 新建会话

- 点击左上角 **烧瓶图标**（实验会话）：创建实验设计会话，自动关联舱体
- 点击左上角 **书本图标**（知识问答）：创建通用问答会话

#### 会话列表

- 实验会话：显示舱体图标 + 运行状态（运行中 / 设计中 / 已完成）
- 知识会话：显示书本图标
- 锁定会话（实验进行中）：显示锁图标，不可关闭

#### 活跃实验追踪

左侧栏底部显示所有运行中的实验，点击可切换到对应会话。

### 3.3 AI 智能助手对话

#### 3.3.1 发送消息

- 在底部输入框输入问题，按 **Enter** 发送（Shift+Enter 换行）
- 支持多轮上下文对话
- 流式输出，逐字显示

#### 3.3.2 检索模式

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| `mix` | 图+向量融合检索（推荐） | 需要 RAG 生成答案 |
| `naive` | 纯向量搜索 | 简单文档检索 |
| `local` | 实体+关系检索 | 特定实体相关问题 |
| `global` | 社区摘要检索 | 广泛知识关联 |
| `hybrid` | local+global 组合 | 综合查询 |
| `bypass` | 直连 LLM | 无需检索的纯对话 |

当前配置在 API 端设置（`/query/stream` 调用时指定 `mode`）。

#### 3.3.3 思考折叠

- LLM 的 `<<think>>...</<think>>` 思考内容在流式输出期间显示为蓝色折叠面板
- 默认收起，显示「思考过程 (点击展开)」
- 点击展开查看完整思考内容
- 流结束后所有思考块保持折叠状态

#### 3.3.4 本地指令（无需 LLM）

直接在输入框输入以下指令即可触发：

| 指令 | 功能 |
|------|------|
| `启动[舱名]` | 启动指定实验舱（如：启动燃烧科学舱） |
| `终止[舱名]` | 停止指定实验舱 |
| `终止任务` / `停止` | 停止所有运行中的实验 |
| `查询状态` | 查看所有舱体状态汇总 |

### 3.4 实验步骤设计器

#### 打开方式

1. 点击输入框下方的 **「打开实验步骤设计器」** 按钮
2. AI 对话返回 DAG 步骤后自动展开

#### 设计器功能

- **添加步骤**：点击「添加步骤」或两步骤之间的 `+` 按钮
- **编辑步骤**：展开步骤卡片，编辑名称、说明、执行目标、仪器参数、前提条件
- **并行分组**：设置 parallelGroup，同组步骤可并行执行
- **DAG 预览**：显示横向流程图，蓝色实线=串行，绿色虚线=并行
- **预览计划**：查看完整执行计划 Markdown
- **重新生成描述**：将修改后的 DAG 发回 AI 重新生成描述
- **生成执行计划**：将 DAG 序列化为 Markdown 计划
- **开始执行**：打开确认对话框，确认后启动实验

#### 开始执行流程

1. 点击「开始执行」→ 确认对话框显示步骤概览
2. 点击「确认执行」：
   - 当前对话被锁定
   - 活跃实验会话切换到监控模式
   - 大屏对应舱体 DAG 更新为新步骤
   - 告警日志记录「实验启动」事件
   - 左侧栏「活跃实验」显示进度

### 3.5 文档管理

点击顶部「上传文档」按钮，支持：

- PDF / TXT / Markdown 等格式
- 批量上传
- 上传进度实时显示
- 处理状态追踪（pending → processing → processed / failed）

上传后文档自动进入 RAG 索引流程，完成后可通过 AI 助手进行知识问答。

### 3.6 紧急终止

点击顶部「紧急终止」按钮，立即停止所有运行中的实验，所有舱体切换为待机状态，3 秒后自动解除紧急模式。

## 4. 双屏联动示例

### 场景：设计并执行一个燃烧实验

1. **平板端**：新建实验会话，输入「设计一个壬烷液滴微重力燃烧实验」
2. **AI 响应**：返回实验方案描述 + DAG 步骤（自动嵌入消息下方）
3. **调整参数**：在 DAG 编辑器中修改点火温度参数
4. **开始执行**：点击「开始执行」→「确认」→ 对话锁定
5. **大屏响应**：
   - 燃烧科学舱状态变为「运行中」
   - DAG 步骤开始依次推进（蓝色脉冲节点）
   - 告警日志显示「实验启动」和每个步骤的完成/失败事件
   - 进度条实时更新
6. **平板追踪**：左侧「活跃实验」显示当前步骤名称
7. **完成**：大屏显示「已完成」，告警日志记录「实验全部完成」

## 5. 故障排查

### 5.1 页面无法加载

- 确认 API 服务器运行在 `localhost:9621`
- 确认前端开发服务器运行在 `localhost:5173`
- 检查浏览器控制台是否有 CORS 错误

### 5.2 AI 无响应

- 检查 `.env` 中 LLM API Key 是否有效
- 检查 LLM 服务地址是否可访问
- 确认知识库中有已处理的文档（上传文档后等待处理完成）

### 5.3 思考折叠显示异常

- 确认 LLM 输出使用了 `<<think>>...</</think>>` XML 标签包裹思考内容
- 思考标签在 Markdown 正文内显示 → 重启前端开发服务器

### 5.4 DAG 不显示

- 确保实验会话中有 `dagSteps` 数据（通过 AI 解析或手动设计）
- 确认舱体关联正确（实验会话关联对应舱体 ID）

## 6. 配置说明

### 6.1 环境变量配置

所有配置通过项目根目录的 `.env` 文件管理（无需 config.ini）：

```bash
# 服务
PORT=9621

# LLM（示例：SiliconFlow OpenAI 兼容服务）
LLM_BINDING=openai
LLM_BINDING_HOST=https://llm.actscal.org/v1
LLM_BINDING_API_KEY=your-api-key
LLM_MODEL=lab-chat

# Embedding
EMBEDDING_BINDING=openai
EMBEDDING_BINDING_HOST=https://api.siliconflow.cn/v1
EMBEDDING_BINDING_API_KEY=your-embedding-key
EMBEDDING_MODEL=BAAI/bge-m3
EMBEDDING_DIM=1024

# Reranker（可选，推荐）
# RERANK_BINDING=cohere
# RERANK_MODEL=BAAI/bge-reranker-v2-m3
```

> **重要**：更换 Embedding 模型后必须清空 `rag_storage/` 目录重新索引。

### 6.2 快速启动

```bash
# 1. 激活虚拟环境
source .venv/bin/activate

# 2. 启动 API 服务器
lightrag-server

# 3. 启动前端（另一个终端）
cd lightrag_webui
npm run dev
```

访问 `http://localhost:5173/webui/#/spacelab` 进入系统。
