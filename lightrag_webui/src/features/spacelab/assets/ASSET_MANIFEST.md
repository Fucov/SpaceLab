# SpaceLab 3D Asset Manifest

This directory is reserved for legally usable 3D assets.

- `models/`: self-authored or properly licensed glTF/GLB instrument assets.
- `textures/`: authored or CC0 PBR maps for metal, glass, plastic, screens, liquids, and benchtop surfaces.
- `hdri/`: authored or CC0 HDR environments.

Current implementation uses self-authored parametric React Three Fiber instrument assemblies and a generated studio environment. No third-party model, texture, or HDRI files have been imported yet, so there are no external license obligations.
